# 项目总览（AIChat 基于 HuLa）

## 一句话
基于 HuLa 开源体系（二次开发）构建企业级 AI 即时通讯系统，交付四块核心能力：客户端、管理端、服务端、aichat-plugins 通讯插件。

## 交付边界
- 客户端：IM 用户侧能力（消息、会话、AI对话入口等）
- 管理端：运营与后台配置
- 服务端：账号、权限、IM、AI能力、网关与微服务治理
- aichat-plugins：负责 aichat 与各种 openclaw 变体的通讯插件

## 代码仓结构

根目录：`/home/hua/projects/aichat`

| 子项目 | 路径 | 说明 |
|--------|------|------|
| HuLa | `HuLa/` | 即时通讯客户端（Vue3 + Vite7 + Tauri + TS） |
| HuLa-Admin | `HuLa-Admin/` | 后台管理端（Vue3 + Vite + TS） |
| HuLa-Server | `HuLa-Server/` | 后端微服务（Spring Cloud 2024 + Spring Boot3 + Java21 + Maven） |
| aichat-plugins | `aichat-plugins/` | aichat 通讯插件（https://github.com/hua0424/aichat-plugins.git） |

## 核心仓库现状
- 四仓均为有效 Git 仓库
- 分支：`master`
- 远程：`origin` 均可用

## 关键子目录

### HuLa
- `src/`：前端主业务
- `src-tauri/`：桌面容器与 Rust 相关
- `docs/`：原项目文档
- `skills/`：扩展能力

### HuLa-Admin
- `src/`：管理端主业务
- `config/`：构建/环境配置
- `tests/`：测试代码

### HuLa-Server
- `luohuo-cloud/`：业务微服务集合（im/oauth/gateway/ai/...）
- `luohuo-util/`：基础能力与公共组件
- `docs/`：服务端文档
